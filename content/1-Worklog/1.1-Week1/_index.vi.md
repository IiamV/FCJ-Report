### Thành tựu Tuần 1

- Hiểu được các kiến thức cơ bản về AWS và mục đích của các nhóm dịch vụ chính, bao gồm:
  - Compute  
  - Storage  
  - Networking  
  - Database  

- Thành công tạo và cấu hình tài khoản AWS Free Tier.  
- Làm quen với AWS Management Console và học cách điều hướng cũng như truy cập các dịch vụ AWS thông qua giao diện web.  
- Cài đặt và cấu hình AWS CLI, bao gồm:
  - Access Key  
  - Secret Key  
  - Default Region  

- Sử dụng AWS CLI để thực hiện các thao tác cơ bản như:
  - Kiểm tra thông tin tài khoản và cấu hình  
  - Liệt kê các region AWS khả dụng  
  - Xem thông tin liên quan đến EC2  
  - Tạo và quản lý key pair  
  - Kiểm tra trạng thái các dịch vụ đang chạy  

- Có kinh nghiệm thực hành quản lý tài nguyên AWS bằng cả AWS Management Console và AWS CLI song song, xây dựng nền tảng vững chắc cho các tuần tiếp theo.
