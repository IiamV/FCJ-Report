# Summary Report: “DevOps on AWS Workshop”

### Event Overview
**Date:** November 17, 2025  
**Location:** AWS Office  
**Duration:** 8:30 AM – 5:00 PM  
**Audience:** Cloud engineers, DevOps practitioners, and students exploring AWS DevOps tooling.

### Event Objectives
- Establish a strong DevOps mindset in cloud-native environments.  
- Provide hands-on exposure to AWS CI/CD, container services, and IaC.  
- Enable participants to adopt automation, observability, and modern deployment strategies.

### Speakers
- AWS Solutions Architects and the training facilitation team.

---

## Key Highlights

### 1. DevOps Mindset & Culture
- Overview of DevOps values, collaboration patterns, and continuous improvement.  
- Integration of AI/ML practices into DevOps workflows.  
- Introduction to industry-standard performance indicators: **DORA metrics**, MTTR, and deployment frequency.

### 2. CI/CD on AWS
- Source control with **AWS CodeCommit** and effective Git branching strategies (GitFlow, Trunk-based).  
- Pipeline automation using **AWS CodePipeline**.  
- Build & test configuration in **AWS CodeBuild**, including automated testing stages.  
- Deployment patterns with **CodeDeploy** such as Blue/Green, Rolling, and Canary deployments.  
- **Demo:** End-to-end CI/CD pipeline deployment.

### 3. Infrastructure as Code (IaC)
- Designing infrastructure using **AWS CloudFormation** templates, stacks, and drift detection.  
- Building reusable constructs using **AWS CDK** with multiple language options.  
- **Demo:** Deploying sample architectures using both CloudFormation and CDK.  
- Decision guidelines for selecting between IaC tools.

---

## Afternoon Highlights

### 4. Container Services on AWS
- Docker fundamentals and microservices containerization.  
- Managing container images with **Amazon ECR** (scanning, lifecycle policies).  
- Deployment and orchestration with **Amazon ECS** and **Amazon EKS**.  
- Simplified deployment with **AWS App Runner**.  
- **Case Study + Demo:** Comparing microservices deployment methods across ECS, EKS, and App Runner.

### 5. Monitoring & Observability
- CloudWatch metrics, logs, dashboards, and alarms.  
- Distributed tracing with **AWS X-Ray**.  
- **Demo:** Full-stack observability configuration.  
- Best practices for alerting, dashboard design, and on-call readiness.

### 6. DevOps Best Practices & Case Studies
- Deployment strategies: feature flags, A/B testing.  
- Automated testing within CI/CD pipelines.  
- Modern incident management workflows and postmortem processes.  
- Case studies highlighting startup and enterprise DevOps transformation journeys.

---

## Event Experience
The workshop provided a structured, hands-on learning environment with active participation from engineers and students.

**What stood out:**
- **Practical Application:** Real-world CI/CD and containerization examples.  
- **Technical Depth:** Detailed demonstrations spanning from IaC to observability.  
- **Modern Patterns:** Emphasis on automation, security, and reliability within DevOps workflows.

> **Next Steps:** Future sessions will expand into advanced container orchestration, DevSecOps integration, and optimized cloud operations.
